Projects

  • Open RAN-Based 5G Network Simulation

    Simulated virtualized 5G network environments using ZeroMQ, srsUE, gNB, Open5GS Core, O-RAN Near-RT RIC, KPI-MON xApp, and CU-DU split configurations. Explored Open RAN, RAN deployment, UE/base station interactions, and data flow in controlled environments.

  • Covid-19 Prediction in Bangladesh Using Deep Learning Algorithm

    Forecasted the number of daily COVID-19 cases occurring in Bangladesh using various machine learning models like Fbprophet, LSTM, and MLP, and comparing their performances.

  • IoT Based Remote Heart Rate Monitoring System

    Developed an ECG monitoring system using IoT, where heart rate data was processed via Arduino and MATLAB, then transmitted to the IoT cloud and notified to the concerned person via email.

  • Gas Leak Detection and Explosion Prevention with Air Quality Monitoring System

    Designed an ultimate engineering solution against gas leak explosions by detecting gas leaks and taking preventive measures such as limiting them and alerting the user via a mobile phone call. Air quality measurements were also taken.

  • Intelligent Gesture Based Security System

    Designed a contactless security system that is unlocked by means of a stored hand gesture pattern. The simulation was performed using MATLAB, VSPE, Arduino, and Proteus.

  • 4-bit Calculator

    Designed separate circuit blocks in both hardware and Proteus for performing operations: encoding, enabling, addition, subtraction, multiplication, division, and decoding. These blocks were integrated to obtain a fully functional calculator.

  • Stepper Motor Control using MATLAB, SIMULINK, PROTEUS, and ARDUINO

    It involves selecting a suitable motor driver and linking it to the microcontroller. A control program is then created and the motor is fine-tuned. The final step is to execute the program to operate the motor.
